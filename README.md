## Usage
### 1. Importing the Library
```from gogomanga import get_search, get_detail, get_chapter```

### Example 

Search Example
```print(get_search(query="naruto"))```

Detail Example
```print(get_detail(mangaid="boruto", page=1))```

Details Example
```print(get_chapter(mangaid="boruto", chapternum=1))```
